WordList is a program that reads any file, and outputs the containing
words, and their frequency to console. words are organized by
frequency and then alphabetically.

-------------------------------------------------------------------------------

To generate an executable file. use the make command
$ make

This generates the executable file "WordList"
In this folder is several sample documents to be read by the 
To read a file and print its WordCount use the command
$ ./WordList filename.txt

-------------------------------------------------------------------------------

To execute the test file associated with this project. use the command
$ make test

This generates the executable file "test"
To execute this file use the command
$ ./test
